# 📘 LifeSync (HabitClock) – Smart Productivity & Health Tracker

---

## 🚀 Project Overview

LifeSync, also referred to as HabitClock, is a modern, lightweight productivity and health tracking application designed to help users organize their daily routines efficiently while maintaining a healthy lifestyle. The application combines three essential components—task management, time tracking, and fitness monitoring—into a single unified interface.

In today’s fast-paced digital environment, individuals often struggle to balance productivity with health. Long working hours, excessive screen time, and lack of structured routines can negatively impact both physical and mental well-being. LifeSync aims to solve this problem by offering a simple yet effective tool that encourages users to stay organized, mindful, and active throughout the day.

This project is built as a **Minimum Viable Product (MVP)**, focusing on core functionality without involving complex backend systems, authentication mechanisms, or external integrations. The application is designed to work efficiently without requiring internet connectivity, making it fast, reliable, and easy to use.

---

## 🎯 Objectives

The primary objectives of the LifeSync application are:

* To provide a centralized platform for managing daily tasks
* To improve time awareness through a real-time clock system
* To promote healthy habits such as hydration and physical activity
* To deliver a clean and intuitive user experience
* To build a scalable foundation for future enhancements
* To demonstrate full-stack thinking with a focus on UI/UX design

---

## 🧩 Key Features

### 1. 📝 To-Do List Management

The To-Do module allows users to efficiently manage their daily tasks.

**Features:**

* Add new tasks instantly
* Mark tasks as completed
* Delete tasks when no longer needed
* Display tasks in a structured list
* Real-time UI updates upon interaction

This module helps users stay organized and ensures that important tasks are not forgotten.

---

### 2. ⏰ Real-Time Clock System

The clock module is one of the core components of the application.

**Features:**

* Live digital clock updated every second
* Current date display (day, month, year)
* Time-based greeting system (Morning, Afternoon, Evening)
* Option to toggle between 12-hour and 24-hour formats

This feature enhances time awareness and improves productivity by keeping users aligned with their daily schedule.

---

### 3. ⏳ Timer & Stopwatch

The application includes built-in tools for time tracking.

**Timer Features:**

* Set custom countdown duration
* Start, pause, and reset timer
* Visual countdown display

**Stopwatch Features:**

* Start and pause tracking
* Reset functionality
* Accurate elapsed time display

These tools are particularly useful for study sessions, workouts, and focus techniques like Pomodoro.

---

### 4. 🏋️ Fitness Tracker (Manual)

The fitness module encourages users to maintain a healthy lifestyle.

**Features:**

* Track daily water intake
* Track daily step count manually
* Display progress toward daily goals
* Reset daily data

This module promotes habit-building without requiring external devices or sensors.

---

### 5. 📊 Dashboard

The dashboard serves as the main interface of the application.

**Features:**

* Displays current time and date
* Shows active tasks
* Displays fitness summary
* Provides quick access to all modules

The dashboard is designed to be minimal, clean, and highly informative.

## 🏗️ System Architecture

LifeSync follows a simple modular architecture:

* Each feature is implemented as an independent module
* Modules are integrated into a single dashboard
* State is managed locally within the application
* No backend or external API dependencies

This architecture ensures simplicity, speed, and ease of development.

---

## 🔒 Privacy Policy

LifeSync prioritizes user privacy and data security. The application does not require user authentication or collect any personal information. All data is stored locally on the user’s device and is never shared with external servers or third parties. The app operates fully offline and does not use tracking technologies. Users have full control over their data and can delete or reset it at any time. This ensures a safe and private user experience.

## 🔄 Workflow

1. User opens the application
2. Dashboard loads with current time and date
3. User interacts with:

   * To-Do list (add/remove tasks)
   * Fitness tracker (update water/steps)
   * Timer or stopwatch
4. All changes are reflected instantly
5. User continues managing daily routine efficiently
